# 🌊 Beachdays

**Beachdays** is a modern, beachwear-inspired web project built with **Tailwind CSS**.  
It focuses on a clean, premium summer aesthetic with soft gradients, parallax visuals, and smooth animations.

---

## ✨ Features

- 🎨 Custom Tailwind theme (colors, fonts, gradients)
- 🌅 Beach-inspired background gradient
- 🌀 Smooth animations (slow spin, parallax-ready)
- 📱 Fully responsive design
- ⚡ Fast and lightweight setup
- 🧼 Clean, scalable utility-first styling

---

## 🛠 Tech Stack

- **HTML5**
- **Tailwind CSS**
- **JavaScript (ES6)**
- **Node.js** (for Tailwind build process)

---

## 🎨 Design Theme

Beachdays uses a **Warm Sand Gradient** for a premium summer feel:

- Soft ivory
- Warm sand
- Subtle blush tones

---

## Getting Started

### Install dependencies
- npm install

### Build Tailwind CSS
- npx tailwindcss -i ./src/css/styles.css -o ./output.css --watch
